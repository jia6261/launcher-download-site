## 使用您的网站

### 1. 浏览启动器功能

访问首页，您可以看到启动器的核心功能介绍，包括自动下载、离线模式、双版本支持等。点击"功能"链接可以快速跳转到功能介绍部分。

### 2. 下载启动器

点击导航栏中的"下载"链接进入下载页面。选择适合您操作系统的版本：

- **Windows**: 提供 64 位和 32 位版本
- **macOS**: 提供 Intel 和 Apple Silicon 版本
- **Linux**: 提供 AppImage 和 Snap 两种格式

点击"下载"按钮即可获取安装程序。

### 3. 查看详细教程

点击导航栏中的"教程"链接进入教程页面。您可以找到：

- **快速开始**: 四个简单步骤快速上手
- **功能详解**: Java 版启动、基岩版启动、离线模式、版本管理等详细说明
- **常见问题**: 解答常见的使用问题
- **高级设置**: JVM 参数配置等高级选项
