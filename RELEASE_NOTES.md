# Release Notes - Beta 1.0.0

## 版本信息

**版本号**: 1.0.0-beta.1  
**发布日期**: 2025-11-05  
**状态**: Beta（测试版）

## 新增功能

### 核心功能
- ✅ Minecraft 启动器下载和教程网站
- ✅ 支持 Java 版和基岩版启动器
- ✅ 完整的模组管理系统
- ✅ AI Builder 1.21 模组集成

### 网站页面
- ✅ **首页** - 启动器功能介绍和快速开始指南
- ✅ **下载页** - 多平台启动器下载（Windows、macOS、Linux）
- ✅ **教程页** - 详细的使用教程和常见问题解答
- ✅ **模组页** - Mod 库和 AI Builder 模组详细说明

### 技术特性
- ✅ 现代化响应式设计（Tailwind CSS 4）
- ✅ React 19 + TypeScript 构建
- ✅ shadcn/ui 组件库集成
- ✅ 完全响应式，支持移动端、平板和桌面
- ✅ 快速加载和优化的性能

## 已知问题

### 当前限制
1. **模组下载** - 目前仅包含 AI Builder 1.21 模组，后续将添加更多模组
2. **启动器文件** - 下载链接为占位符，需要替换为实际的启动器可执行文件
3. **用户认证** - 暂未实现用户登录和账户系统
4. **在线功能** - 暂未实现在线服务器列表和多人游戏功能

## 安装和使用

### 前置要求
- Node.js 18+
- npm 或 pnpm

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/jia6261/launcher-download-site.git
cd launcher-download-site

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```
